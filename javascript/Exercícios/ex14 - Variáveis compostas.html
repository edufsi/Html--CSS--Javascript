Variáveis Compostas

Variáveis compostas/ vetores/ arrays são variáveis que possuem vários elementos. Cada elemento é composto por seu valor e por uma chave de identificação.

Listas

Para fazer uma lista, basta usar colchetes [] ao declarar a variável

var a = []

let a = [231, 346, 1, 987]

Adicionando valores

Podemos fazer isso atribuindo diretamente um valor à variável, desse jeito:

let b = [10, 20, 30]

b[3] = 40

b passará a ser [10, 20, 30, 40]

Também podemos fazer isso com o método PUSH, que insere um valor novo à última posição da lista. let b = [10, 20, 30] b.push(40) b passará a ser [10, 20, 30, 40]

length

O atributo length retornará o número de elementos em uma lista

let b = [10, 20, 30, 40]

b.length retornará o valor 4

sort

Método que ordena todos os valores numéricos em ordem crescente ou todos os valores string em ordem alfabética.

indexOf

método de listas que toma como parâmetro um valor e retorna o índice da primeira ocorrência desse valor. Caso não haja ocorrência do valor dado no vetor, retorna -1

a = [1, 2, 3, 4, 5]

a.indexOf(1) -> 0

a.indexOf(2) -> 1

a.indexOf(4) -> 3

a.indexOf(38) -> -1
