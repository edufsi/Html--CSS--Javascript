Tratamento de erros

Erros são objetos com uma descrição de algo que deu errado. São chamados de exceções.

TRY ... CATCH ... FINALLY

Para lidar com erros em Javascript, uma das formas é usando try ... catch.

O bloco try tenta ser executado.

Caso alguma exceção seja lançada no bloco try, o bloco catch será executado. Catch pode ou não estar associado a uma exceção, ou seja, podemos criar vários blocos catch para diferentes exceções.

Finally será executado, independentemente do resultado.

try {
    // Qualquer coisa
}
        
// Se o erro1 acontecer:

catch(erro1) {
    console.log(erro1)
}
        
//Se o erro2 acontecer:

catch(erro2) {
    console.log(erro2)
}
        
//Se qualquer outro erro acontecer:

catch {
    console.log('ERRO')
}

finally {
    //Qualquer coisa 
}

TROW

Levanta uma exceção customizada.

Digamos, por exemplo, que queremos que um usuário digite um número:

try {
    let x = window.prompt('Digite um número')
    if (isNaN(x)) {
        throw `${x} não é um número`
    }
    
    if (x == '') trow `O usuário não digitou nada`
}
catch(erro) {
    console.log(erro)
}

Se o usuário não digitar um número, estamos levantando uma exceção e escrevendo ela.
