Gerador aleatório de números

Método Math.random()

Esse método gera um número aleatório ente 0 e 1. Para fazer um gerador de números aleatórios com outros números, devemos multiplicar o valor obtido por um número (o número máximo do intervalo do qual queremos tirar um número aleatoriamente. Se estivermos falando de um intevalo de números inteiros, é preciso arredondar o valor com Math.floor. Para gerar um intervalo diferente de 0 até o número pelo qual estamos multipicando Math.random, devemos somar o valor que queremos como mínimo. Assim, a fórmula fica asssim:

Math.floor(Math.random() * o número máximo que você quer no intervalo) + o número mínimo que você quer no intervalo
