Exemplo de caixa box-level

Parágrafos são exemplos de caixas box-level, mas links são exemplos de caixas inline-level

Podemos tansformar elementos inline em box-level e vice-versa nas CSS usando display: inline ou display: block, ou até mesmo display: inline-block

Ou seja, podemos transformar até mesmo links em caixas box-level
