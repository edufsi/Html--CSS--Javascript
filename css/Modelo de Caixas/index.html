Modelo de Caixas

Quando temos um documento HTML, tudo aquilo que é exibido na tela é exibido em forma de caixa, e grande parte das caixas em HTML podem ser colocadas dentro das outras, formando uma hierarquia de caixas. O conceito de colocar uma caixa dentro da outra é chamado de "aninhamento".

Um parágrafo, por exemplo, nadda mais é do que uma caixa com texto.

Elementos da caixa

Toda caixa possui duas dimensões: height (altura) e width (largura)

Toda caixa possui uma borda, a linha que circunda a caixa. O espaço vazio entre a borda e o conteúdo é chamado de padding (acolchoamento/preenchimento). O espaço vazio externo da caixa é chamado de margin (margem). Dentro da margem e grudada na borda, ou seja, fora da caixa, temos a outline (contorno).

Tipos de caixa

  1. Box-Level

    2. Sempre se inicia em uma linha nova e sempre ocupa a largura inteira da página.

    Exemplo: tags div, p, main, h1, h2, h..., header, video.

  2. Inline-Level

    3. Não pula uma linha, apenas pula para o lado e a caixa é exatamente do tamanho necessário para o conteúdo, ou seja, a caixa não ocupa toda a largura da página.

    Exemplo: tags span, a, code, strong, small
