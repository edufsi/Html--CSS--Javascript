Position

A position é uma propriedade CSS que define como um elemento qualquer pode ser posicionado no corpo do documento. Ela vem acompanhada de 4 outras propriedades: top, bottom, right e left, que vão determinar a localização final do objeto, deslocando o elemento conforme o especificado. Tipos de Position

Vamos com os exemplos:

Static

É a posição padrão. Ela não tem nada especial, ou seja, ela segue o fluxo da página.Por isso, ela não é afetada pelos elementos top, bottom, left, right.

Essa é a div mãe

Essa é a div filha

Fixed

Um elemento com position: fixed faz o elemento ficar fixo na página, sem se importar com a scrollbar. Ele fica fixo porque é posicionado em relação à viewport, ou seja, à área visível para o usuário da página web.

Essa caixa está com position fixed nesse canto da página.

Sticky

O elemento fica fixo na tela a partir do momento em que a barra de rolagem alcançá-lo.

Essa caixa vai se juntar na caixa do exemplo acima

Relative

Um elemento com position: relative o faz ficar posicionado em ralção à sua posição original. Ou seja, top, bottom, right e left deslocarão o elemento a partir da posição normal dele, a posição estática

Absolute

Posicionado em relação ao elemento mais próximo dele que possui um position: relative. Esse elemento precisa estar dentro de uma hierarquia com um elemento com um position relative. Caso não esteja, o position: absolute será absoluto ao corpo da página.
