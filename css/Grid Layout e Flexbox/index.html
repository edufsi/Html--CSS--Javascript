Após colocado display:flex, perceba que os itens dentro da div se alinharam. Isso o corre porque o flex posiciona os itens sobre dois eixos: o eixo princiapal, que é o horizontal que aponta da esquerda para a direita, e o eixo cruzado, que é vertical e aponta de coma para baixo. Podemos inverter isso com a propriedade flex-direction. Essa propriedade aceita:

  • row: definido por padrão. Faz com que os blocos dentro docontainer se alinhem em uma linha da esquerda para a direita
