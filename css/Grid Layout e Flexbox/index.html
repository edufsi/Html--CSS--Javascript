Após colocado display:flex, perceba que os itens dentro da div se alinharam. Isso o corre porque o flex posiciona os itens sobre dois eixos: o eixo princiapal, que é o horizontal que aponta da esquerda para a direita, e o eixo cruzado, que é vertical e aponta de coma para baixo. Podemos inverter isso com a propriedade flex-direction.

Justify Content

Define o alinhamento dos itens de acordo com o eixo principal, ou seja, depende do flex-direction

  • flex-start: alinha ao início do eixo principal. Se temos que o eixo principal é o horizontal e ele vai da esquerda para a direita (padrão), então flex-start colocará os itens à esquerda. Agora, se tivermos com flex-direction: row-reverse, o eixo principal será o horizontal da direita para a esquerda, o que significa que os elementos serão colocados à direita.

Align-content aceita:

  • flex-end, flex-start, center, stretch, todos os elementos que align-items aceita
  • space-around: comportamento idêntico ao align-items: center. Separará os elementos por espaços iguais, centralizando-os no eixo secundário
  • space-between: também distribui os elementos no centro do eixo secundário, porém gruda elementos no flex-start e no flex-end
