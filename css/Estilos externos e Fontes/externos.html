Estilos CSS externos

Para fazermos estilos externos, precisamos linkar dentro da parte head do nosso documento o documento em html com um documento css através da tag link, utilizando os elemntos rel="stylesheet" e href para o documento.

Essa é a forma mais eficiente, visto que estilos in-line são muito específicos e estilos internos se aplicam apenas a um documento.
