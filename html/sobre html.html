HTML

HTML = Hypertext markup language, Linguagem de marcação hipertexto

A HTML não é uma linguagem de programção, como o próprio nome diz ela é uma linguagem de marcação, ou seja, ela não funciona com variáveis, iterações, condições, objetos, etc. Seu funcionameto é baseado nas marcas ou etiquetas (tags, em inglês)

Tags

Uma tag é um conjunto de palavras entre sinais de colchete angular (<>). O exemplo a seguir cria um parágrafo e coloca o texto entre as tags no parágrafo

<p> Lorem ipsum dolor sit amet, consectetur adipisicing elit. Repellendus quia officiis tempore sit voluptatibus? Quos minima, in architecto optio maiores magnam dicta molestias? Pariatur eum quas reprehenderit et quis in. </p>
