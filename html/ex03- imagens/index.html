Formatos de imagens

São dois os principais formatos de imagem: jpeg e png.

O algoritmo de compactação jpeg é usado para compactar imagens, ou seja, gerar imagens com um tamanho muito reduzido.

O formato Portable Network Graphics (png) foi desenvolvido pelo W3C com o objetivo de substituir o gif. É um formato também compactado, mas não tanto quanto o jpg. Sua principal característica, e o que diferencia ele do outro, é a capacidade de configurar a opacidade de cada pixel

Tamanho de imagens

Para a criação de um site, é muito importante utilizar arquivos com o menor tamanho possível. Nada de colocar uma imagem com uma resolução gigante e depois tentar diminuí-la com código- ela não ficará mais leve dessa maneira. Por isso, é preciso deixar a imagem com o tamanho certo

Ícone de Favoritos- Favicom

São aqueles pequenos ícones que aparecem do lado do título de uma página da web

Para usar uma favicom, é preciso ter o ícone em formato ico (de preferência), png ou svg (o svg seria a melhor extensão, pois é vetorizado- "Arquivos em vetor são muito mais flexíveis e dependem de equações matemáticas para gerar imagens com qualidade perfeita"-, não fosse o fato de nem todos os navegadores suportarem esse formato.
