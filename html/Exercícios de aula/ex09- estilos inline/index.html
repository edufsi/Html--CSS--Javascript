Estilos in-line

Estilos in-line referem-se as CSS inseridas diretamente no body de seu documento html. Elas são inseridas através do elemento style dentro das tags. Essa é a forma mais específica de colocar estilos nos seus sites. Ela deve ser evitada ao máximo, visto que inserindo o estilo junto com o conteúdo deixa a leitura do código difícil e deixa a manutenção muito difícil e trabalhosa (caso uma alteração seja feita será necessário alterar todo o documento.
