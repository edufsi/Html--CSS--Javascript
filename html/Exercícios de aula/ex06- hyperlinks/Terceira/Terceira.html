Navegando por pastas locais: servidores web normalmente rodam o sistema operacional Linux. Nesse sistema, se estiver se referindo à pasta atual do servidor, pode-se usar ./ antes do nome do arquivo. Agora, se quiser se referir à pasta imediatamente superior na hierarquia, use ../ para voltar para a pasta mais externa.

No exemplo acima, utilizamos ../ para voltar a página mais externa, usamos / para entrar na pasta "Segunda" e então / para selecionar dos itens da página "Segunda" o arquivo Segunda.html
