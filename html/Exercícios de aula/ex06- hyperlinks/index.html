Hyperlinks

O que são hyperlinks?

Hyperlinks são textos que permitem a ligação de um ponto a outro na World Wide Web

Como criar hyperlinks?

Para criar hyperlinks precisamos criar âncoras através da tag <a>. O principal atributo da tag a é href, que pega uma referência hypertext.

Exemplo:

<a href="https://github.com/edufsi">Link para meu Github</a>

O resultado é esse: Link para meu Github

Outro atributo importante da tag a é o hreflang, que determina a língua que o navegador deve esperar encontrar no seu site.

Destino do site aberto

Como padrão, o site aberto irá sobrepor o site já aberto. Isso pode ser um comportamento desejável se você está navegando por outras páginas de um mesmo servidor, mas quando o link é para uma página terceira, o bom é que se abra uma nova guia quando o site for aberto.

Para isso, usamos o atributo target.

Esse atributo suporta os seguintes valores:

  1. _blank: abre o link em uma nova janela em branco
  2. _self: abre o link no frame atual (padrão)
  3. _top: desfaz os frames e abre o destino no navegador completo
  4. nome-do-frame: caso esteja usando frames, indicar o nome da janela e abrir
  5. _parent: similar ao uso do_top em uma referência à janela mãe

Como o uso de frames é raro, por agora é bom saber apenas sobre o _self e o _blank

Exemplo:

<a href="https://github.com/edufsi" hreflang="pt-br" target="_blank">Link para meu Github</a>

Terá como resultado um hyperlink que abrirá uma nova guia no navegador: Link para meu Github

Indicando a natureza do destino

Para indicar a natureza do destino do link, utilizamos o atributo rel

O atributo rel aceita vários valores, entre eles:

  1. next: indica que o link é para a próxima parte do documento atual
  2. prev: indica que o link é para a parte anterior do documento atual
  3. author: indica que é um link para o site do autor do atual artigo
  4. external: indica que é um link para um site que não faz parte do site atual
  5. nofollow: indica o endereço para um site não endossado, como um link pago.

Fazendo dowloads

Para criar um link em que será feito um dowload, utiliza-se os atributos dowload e type na tag a.

No atributo download coloca-se o nome do arquivo a se baixar e no atributo type coloca-se o tipo de arquivo que será baixado. O atributo type serve para informar ao seu navegador qual será o tipo de arquivo baixado pelo seu link. Para isso ele aceita valores específicos, que podem ser conferidos no site:
