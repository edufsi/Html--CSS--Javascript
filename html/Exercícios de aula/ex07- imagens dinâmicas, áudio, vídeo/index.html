Sites são acessados em todas as mídias

Antes de tudo, é preciso entender que sites são acessados em muitos tipos de mídias com telas de tamanhos diferentes: celulares, computadores, tablets, televisões. Assim, como já mencionado no curso, se usarmos imagens muito grandes e com muita resolução em nossos sites, deveremos forçar a redução do tamanho para telas menores com CSS, mas isso não diminui o quanto o arquivo pesa, assim nosso site acaba ficando lento. Ninguém gosta de sites lentos, nem mesmo os próprios mecanismos de busca dos navegadores(SEO- (otimização para mecanismos de busca). Dessa forma, precisamos adaptar nosso conteúdo ao tamanho da tela do dispositivo que acessará nosso site, tirando assim peso desnecessário. Essa é a função das imagens dinâmicas

Imagens dinâmicas

Para fazer uma imagem dinâmica, precisamos conhecer as tags <picture> e <source>

A tag picture é usada para dar flexibiliade para os desenvolvedores possam fornecer especificações para a imagem a ser colocada

A tag source é usada para definir muitos recursos de mídia em diferentes formatos. Para colocar uma imagem flexível, precisamos especificar suas características utilizando a tag source com o elemento media, que aceita, entre outros, o max-width: xxxpx, o que dá um valor máximo de largura para se colocar determinada imagem. Além disso, também é preciso especificar o tipo da mídia com o elemento type e a imagem a ser colocada com o elemento scrset.
