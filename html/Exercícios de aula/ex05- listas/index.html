Trabalhando com listas

Listas Ordenadas

Listas ordenadas são, obviamente, aquelas que seguem uma ordem. Essa ordem pode ser por letras, números e números romanos. Fazemos uma lista ordena utilizando a tag OL (ordered list), com os elementos com a tag LI, que tem seu fechamento opcional

  1. Acordar
  2. Tomar café
  3. Ir para a escola
  4. Almoçar
  5. Estudar
  6. Jantar
  7. Dormir

Listas não ordenadas

Utilizam a tag UL (unordered list) e a tag LI para seus elementos

  • Pão
  • Café
  • Margarina
  • Queijo
  • Alface
  • Tomate
  • Arroz
  • Azeite
  • Feijão

Listas de definições

São espécies de "dicionários". Cada elemento da lista tem um título. Usamos a tag DL para criá-la, a tag DT para o título e tag DD para o que não é o título.
