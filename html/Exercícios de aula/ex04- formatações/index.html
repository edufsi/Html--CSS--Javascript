FORMATAÇÕES EM HTML

Foco em semântica, não em forma

É importante destacar que a HTML não deve ser utilizada para mudar a forma de elementos, mas sim para adicionar sentidos, significados. Para alterar a forma, usamos CSS. Por exemplo, tags como <u> (muda texto para itálico), <b> (muda texto para negrito) e <big> (aumenta o tamanho do texto) já são obsoletas, pois elas não tem sentido (não são semânticas), apenas mudam formas, nesse caso de textos.

Texto excluído/deletado

Texto que deve ser lido, mas não considerado

Usamos a tag DEL para fazê-lo, <del>assim</del>

Texto inserido

Indicação de que aquele texto foi inserido (sublinhado)

Usamos a tag INS para fazê-lo, <ins>assim</ins>

Texto sobrescrito

Utilizamos a tag SUP para sobrescrever. Por exemplo: x<sup>20</sup>=0

Texto subscrito

Utilizamos a tag SUB para subscrever. Por exemplo: H<sub>2</sub>O

Código

Para inserir um código através de uma fonte monoespaçada, use a tag CODE.

Exemplo:

 

def primo(num):
    primo = True
    for value in range (2, num):
        if num % value == 0:
            primo = False
    if primo:
        return 1
    else: 
        return int(0)
    
def quantos_primos(intervalo):
    primos = 0
    for num in range(intervalo):
        primos += primo(num)
